how to disable autorun programs from usb stick, to stop some viruses.


install "Update for Windows XP (KB967715)" from:
http://www.microsoft.com/downloads/details.aspx?FamilyID=c7dbcde3-7814-47c5-849e-e64ecfb35d74&displaylang=en

(for other versions of windows: http://support.microsoft.com/kb/967715)

Windows XP Professional

-Click Start
-click Run
-type Gpedit.msc , press enter
-navigate: Computer Configuration/Administrative Templates/System.

double click "Turn off Autoplay"
enabled; all drives

same for User Configuration/Administrative Templates/System

also from: (for other versions of windows: http://support.microsoft.com/kb/967715)


NOTE: you can hold "SHIFT" when insert usb-stick. This should stop autorun too.
