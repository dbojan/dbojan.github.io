windows xp, create html page with images

Download it, unzip, and put "create html page with images.bat" 
in folder with png or jpg files. It will create/overwrite index.html

Edit bat file to change extension of includes images. Change the line to .jpg, .png, or something else.

If you start it from usb memory stick, files will be sorted according to the time of writing.
To fix it, copy dir with files and bat file to hard disk, and start bat file from there.
