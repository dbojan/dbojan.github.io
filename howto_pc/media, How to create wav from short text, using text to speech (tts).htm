How to create wav from short text, using text to speech (tts)

Use http://www.voicerss.org/ for example

Live demo: http://www.voicerss.org/api/demo.aspx

To create wav:
-register for free
-check you api key under 'My profile'

-enter address in this format:
http://api.voicerss.org/?key=1234&src="hello it's me"&hl=en-gb&c=WAV&f=44khz_16bit_mono
Save wav file from browser, either use save option, or right click on media and select 'save'.

You can change format or codec.
Check their documentation: http://www.voicerss.org/api/documentation.aspx


Basic version is free. 350 Daily Requests, limited to 100KB per request.

There are many other programs to create longer text to speech, or you can check https://librivox.org/, books read by people.
