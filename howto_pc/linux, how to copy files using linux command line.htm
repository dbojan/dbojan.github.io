how to copy using linux command line
cp -a /path/source/* /path/destination/

or 

cd /destination
cp -a /path/source/* .

(dot means copy here)

or use some file manager, like pcmanfm, that comes with slitaz linux(50mb iso)


note to copy files and folders only, without permisions use -r instead of -a:
cp -r /path/source/* /path/destination/

or 

cd /destination
cp -r /path/source/* .
