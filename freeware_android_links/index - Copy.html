General apps:
Alarm clock extreme free - set volume to increase slowly, random audio file based on author id tag.
Challenger Viewer - comic book reader, continuous/scrollable vertical views
Dolphin browser - web browser.
Dolphin browser zero delete cache on exit automatically, less customization.
Ebook reader, Adobe reader - pdf readers
ES file explorer - file manager
FBReader + ColorDict Dictionary - book reader. My settings: black background + dark green text.
Goodplayer - can remember audio file position after restarting app. Good for audio books. Also plays mka files.
GPS Logger for Android can do continous GPS logging.
Watchbeep clock beep every hour/half an hour. Make work breaks. 

VOIP - voice over internet protocol - telephone calls/messages/video over Internet:
Voxox Call to fixed lines cheaper. They will top your account up to 1$ each month. Sometimes login does not work.
Csipsimple internet call, messages. No photos, videos require free plugin. Use with sip account provider like http://www.ippi.com/, for example. For PC try Microsip or PhonerLite.
Linphone sip app for desktop and mobile, used for internet calls, video is supported too. If you have problems reaching others, go to 'Settings', and disable ICE.
Android supports audio voip too. See 'settings/accounts'.
There are also proprietary solutions like Skype, WhatsApp or Viber.

Free offline navigation/gps, (using OSM data):
-Osmand - good navigation. You can download version with no limitations, named Osmand~, from here: OsmAnd-nightly-2.3.apk or OsmAnd-nightly.apk. 
Which ever version is newer when starting install. Works with the screen off.
-MAPS.ME –Offline Map & Routing ok, somewhat simplified search. Enter "City name, Street name", if searching for destination in the another location. 
Great routing. Does not work with screen off.
-Here maps, sometimes exits with error, free registration required, (has offline mode after reg.), big downloads.
-Navmii world, navmii usa, ads.
-MapFactor: GPS Navigation (sometimes strange routing?)
-GPS Navigation BE-ON-ROAD Voice guidance does not work after 7 days, unless you buy premium version. On the map of Serbia you have to use Cyrillic keyboard.


Advanced:
Link2SD disable apps temporary. Also can move apps to sdcard.
Scriptmanager create scripts, like to copy random file to alarm.mp3. More info on pc_howto page.
Wifi password recovery Shows wifi password you entered previously on cell phone. Needs root.
